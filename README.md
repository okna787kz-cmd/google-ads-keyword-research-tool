# Google Ads Keyword Research Tool

An internal single-user tool for researching keywords and preparing my own Google Ads campaigns.

## Purpose

The tool is being developed to:

- generate keyword ideas using KeywordPlanIdeaService;
- retrieve historical search volume metrics;
- analyze keyword competition and bid ranges;
- prepare and improve keyword structures for Google Ads campaigns.

## Access and data use

The tool will access only Google Ads accounts owned or administered by its operator.

Google Ads API data will not be sold, resold, published, or shared with third parties.

The project is an internal tool and is not offered as a public service.

## Status

Currently in development.
